## 🌸 How to Install

1. Open **Visual Studio Code**
2. Go to the **Extensions** panel (`Ctrl + Shift + X`)
3. Search for **NightBloom**
4. Click **Install**
5. Open the **Command Palette** (`Ctrl + Shift + P`)
6. Choose `Color Theme` → Select **NightBloom by Rutvi**
